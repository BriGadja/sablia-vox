# Sablia Vox

Voice AI Agent Platform — B2B sales automation via AI voice agents.

**Domain**: [vox.sablia.io](https://vox.sablia.io)

## Quick Start

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # Production build
npm run lint     # ESLint
```

## Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Framer Motion
- **Database**: Supabase (PostgreSQL)
- **State**: React Query
- **Auth**: Supabase Auth
- **Hosting**: Vercel

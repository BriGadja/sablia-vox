# 2026-04-15

## Session Learnings

- **Recharts v3 ghost legend entries**: Recharts v3.2.1 `<Legend content={fn}>` payload includes entries where `entry.value = "value"` and `entry.payload.value = undefined`. Custom legend renderers must filter `entry.payload?.value != null && typeof entry.payload.value === 'number'` before calculating percentages. Fix: `d8fcb75`.
